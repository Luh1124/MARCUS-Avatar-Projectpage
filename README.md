# MARCUS-Avatar-Projectpage

Official project page for MARCUS-Avatar: Monocular Avatar Reconstruction via Cascaded Diffusion Priors and UV-Space Differentiable Shading.

This repository contains a static GitHub Pages site with interactive 3D avatar previews, PBR material maps, method figures, and placeholder links for the paper, code, HuggingFace weights, HuggingFace demo, and arXiv.
